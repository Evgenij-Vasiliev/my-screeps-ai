# Screeps AI: Room E37S37 Lab

Мой модульный ИИ для игры Screeps (ветка **test**).
Этот код предназначен для управления колонией в комнате с контроллером 8-го уровня.

### Основные возможности:

- **Автоматизация**: Спавн крипов по заданному лимиту в `rolesConfig`.
- **Уникальные имена**: Генерация имен на основе роли и игрового времени (`Game.time`).
- **Модульность**: Каждая роль вынесена в отдельный файл в папке `role.*.js`.
- **Очистка памяти**: Автоматическое удаление данных погибших крипов.

### Список ролей:

- **Harvester**: Добыча и заправка Spawn/Extensions/Storage.
- **Upgrader**: Развитие контроллера.
- **Builder**: Строительство (или помощь апгрейдеру).
- **Repairer**: Ремонт дорог и контейнеров (игнорирует стены и рампарты).
